契約書アブストラクター

契約書(PDF・画像・テキスト)をアップロードすると、金額・取引先・勘定科目などを自動抽出して一覧化します。セルは直接編集でき、CSVでダウンロードできます。

設定

キーはこのブラウザ(localStorage)にのみ保存され、Anthropic API 以外には送信されません。 契約書ファイルも抽出のために Anthropic API へ直接送信されるのみで、他のサーバーには保存されません。 共有PCで使ったあとは「キーを消去」を押してください。
抽出結果カードの「転記」ボタンを押すと、このURLを開いてデータを引き渡します(URL末尾の #data= にJSONを付与)。 初期値はサンプルの社内決裁システム(デモ)です。 転記すると稟議書として自動起票されます。空欄にすると転記ボタンは動作しません。

抽出したい項目を自由に定義できます。「+ フォーマットを追加」で入力欄を作り、抽出したい項目を書いてください(1行に1項目がおすすめ)。 選択中のフォーマットに基づいて、「契約書読み込み」タブでの抽出結果が整理されます。フォーマットはこのブラウザに保存されます。
