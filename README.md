# 契約書アブストラクター

契約書(PDF・画像・テキスト)をアップロードすると、Claude API が経理処理に必要な情報を自動抽出し、編集可能な一覧表に整理する Web アプリです。結果は CSV(Excel 対応・UTF-8 BOM 付き)でダウンロードできます。

## 抽出される項目

| 項目 | 説明 |
|---|---|
| 契約書名 | 契約書のタイトル |
| 取引先 | 相手方の会社名・氏名 |
| 契約締結日 / 開始日 / 終了日 | 和暦も YYYY-MM-DD に自動変換 |
| 金額(税抜)/ 消費税 / 金額(税込) | 数値として抽出 |
| 支払形態 / 支払条件 | 一括・月額など / 締め支払条件 |
| 勘定科目 | 契約内容から最適な科目を提案(理由付き) |
| 摘要 | 仕訳の摘要欄に使える要約 |
| 備考 | 抽出に不確実な点があれば注意書き |

## 2つの使い方

| 方式 | 場所 | APIキーの扱い |
|---|---|---|
| **GitHub Pages(静的版)** | `docs/index.html` | 画面で入力し、ブラウザの localStorage にのみ保存。ブラウザから Anthropic API を直接呼び出す |
| **ローカルサーバー版** | `server.js` | サーバーの環境変数 `ANTHROPIC_API_KEY` に設定(キーがブラウザに出ないため、複数人で使うならこちら) |

### GitHub Pages で公開する

1. リポジトリの **Settings → Pages** を開く
2. Source: **Deploy from a branch**、Branch: このブランチ、Folder: **/docs** を選択して Save
3. 数分後に `https://<ユーザー名>.github.io/contract-abstracter/` で公開される
4. ページ上部の入力欄に自分の Anthropic API キーを入力して利用

> ⚠️ 静的版は各利用者が自分のAPIキーを入力する方式です。キーは localStorage に保存され Anthropic API 以外には送信されませんが、共有PCでの利用後は「キーを消去」を押してください。

## ローカルサーバー版のセットアップ

```bash
npm install
export ANTHROPIC_API_KEY=sk-ant-...   # Anthropic APIキー
npm start
```

ブラウザで http://localhost:3000 を開き、契約書をドラッグ&ドロップしてください。

### 環境変数

| 変数 | 必須 | 説明 |
|---|---|---|
| `ANTHROPIC_API_KEY` | ✅ | Anthropic API キー |
| `CLAUDE_MODEL` | — | 使用モデル(デフォルト: `claude-opus-4-8`) |
| `PORT` | — | ポート番号(デフォルト: 3000) |

## 対応ファイル形式

- PDF(スキャン画像のPDFも可・1ファイル30MBまで)
- 画像: PNG / JPEG / WebP / GIF
- テキスト: .txt / .md

## 仕組み

1. ブラウザからファイルをアップロード(複数可・並列処理)
2. サーバーが Claude API(structured outputs)に契約書を渡し、JSON スキーマに沿った抽出結果を取得
3. 結果を表に追加。セルは直接クリックして修正可能
4. 「CSVダウンロード」で指定フォーマットの一覧を出力

## 注意事項

- 抽出結果は AI による読み取りのため、金額・日付は必ず原本と照合してください。不確実な点は「備考」列に表示されます。
- アップロードされたファイルはサーバーに保存されず、抽出処理のために Anthropic API に送信されるのみです。
